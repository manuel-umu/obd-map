# OBD-Map

OBD-Map es una aplicación para radios Android. Sustituye la interfaz del sistema por una pantalla única y fluida que junta dos cosas que normalmente van por separado: **la navegación** y **los datos del coche**.

---

## Funcionalidad principal

- **Navegación sin internet.** El mapa y el cálculo de rutas funcionan completamente offline.
- **Guía como un navegador normal.** Indicaciones giro a giro, con **avisos por voz en español**, distancia a la próxima maniobra y hora estimada de llegada. 
- **Los datos del coche en pantalla.** Conectando un adaptador OBD2 Bluetooth (ELM327), muestra métricas en vivo superpuestas al mapa: **consumo instantáneo**, velocidad, revoluciones y más.

## Pensada para el coche

Todo el diseño gira en torno a un uso real conduciendo: sin menús enrevesados, respuestas rápidas, y prioridad a que el mapa y los datos se vean de un vistazo.

## Tamaño de la aplicación

El desarrollo se está haciendo pensando en la poca memoria RAM de las radios **aftermarket** instaladas en los coches. Actualmente se esta utilizando una con 1GB de memoria RAM.
